How to stay updated with JavaScript versions

02/12/2020
By Maisha Maliha

So I started searching about JavaScript versions. And I found nothing. Because there was no official website for “JavaScript”. I was just stunned how there is no proper nice documentation about JavaScript in ECMA organization website. I was like “what the heck?? Such a big famous language has no official website, where I can see the latest features or latest versions”. I was very mad. I went through ECMA organization’s website again. And then I discovered, that, the documentation of JavaScript is under the name ECMAScript. Yes, the other name of JavaScript is ECMAScript. So it did take a while to understand how things are working in ECMA organization website (ECMA International). Unlike Java, ECMA website is not very well. I am not saying Java oracle website is the best doc but still.

JavaScript is a booming Language. It is very popular and in great demand with high salary. Also updates are coming almost every year. So you have to keep up with language update to be able to become a good JavaScript developer and get hired as well.

So here are the websites you should keep your eyes on to stay up to date with JavaScript latest versions and features and also to learn JavaScript.

For seeing a short table of JavaScript versions check this wiki site:

https://en.wikipedia.org/wiki/ECMAScript#Versions

To learn JavaScript as a beginner with example visit MDN, which is the best website to learn JavaScript as a language. Also you will find almost everything related to JavaScript and the advance stuff too:

https://developer.mozilla.org/en-US/docs/Web/JavaScript

To know about the past, present and future versions of JavaScript visit this site:

https://www.ecma-international.org/memento/tc39-m.htm

To see all the versions in details visit this website:

https://www.ecma-international.org/publications/standards/Ecma-262-arch.htm
