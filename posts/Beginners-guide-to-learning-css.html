Beginners guide to learning CSS

07/10/2020
By Maisha Maliha

If you are into web development, you have to learn CSS. CSS may sound easy, in reality it is, but, can get very complex. From the beginning you must follow good practices in writing CSS codes. In this post I will give you the topics of CSS that you should cover. I will also tell you where you can get them.

CSS stands for Cascading Style Sheet. CSS simply gives style to the HTML elements. As a beginner I recommend learning CSS from the below websites:

  • W3SCHOOLS: Here you will get almost everything. Almost all the topics are covered.
  • Mozilla : This website is very good. One of my favorite. But this is not very beginner friendly.
  • W3 ORG : This is a little advance. It is the official website of CSS.
  • CSS-Tricks: A great website to know about CSS related stuff.

For Beginners

For a beginner the pre requisite to learn CSS is to know HTML. It’s a must. I will try to go serially as to what you should cover one after another in CSS.

  • At first you need to learn to select Tags, elements with certain Class or ID. For example, you select all links in your HTML file in CSS like this:

    Code

    a{ color: red; }
    You select all elements with a class this way:

    Code

    .democlass{ color: blue; }
  • Learn to comment in CSS
  • Learn to style texts. For example, change color, change font size, font family and text alignment.
  • Learn about CSS units.
  • 5. Learn to style paragraph word spacing, line heights, text decoration, text indentation and letter spacing.
  • 6. Learn to add margin, padding, width and height.
  • 7. Learn about Box Model
  • 8. Learn to style background of anything by adding color or image.
  • 9. Learn to style images.
  • 10. Learn about outlines and border and border radius.
  • 11. Learn to style lists
  • 12. Learn to style table

For Mid-level

  • 1. Learn CSS selectors in more
  • 2. Learn about display types
  • 3. Learn how to position things
  • 4. Learn flex box
  • 5. Learn grid
  • 6. Learn about overflow
  • 7. Learn about float
  • 8. Learn to style with Opacity
  • 9. Adding Shadow to things
  • 10. Learn to about Gradients
  • 11. Learn about CSS Animation, transition, transform and @keyframe
  • 12. Learn about responsive design which is by using @media

If you have completed all this. You know enough to build websites front-end. You can now build simple and standard websites front-end.

Now I am going to talk about Advance stuff in CSS. These things are something you have to learn if you want a job. Till now we covered all the basics. Now it’s time for advance stuff.

Advance CSS

  • • Learn the famous CSS Framework called Bootstrap. It is a must. If you want a job you have to learn it. Most jobs require you to know it because it helps in faster coding.
  • • Learn CSS Pre-processor like SCSS, SASS or LESS. Pre-processors are something that helps you code less and more simply. Later your simple code is compiled to normal CSS code.
  • • Learn to you use CSS to draw anything. You can use JavaScript too. Like these : CSS Tajmahal
  • • Learn to do complex CSS Animation using CSS animation libraries or just pure CSS and JavaScript.For example: Mesmerizing or like this: particles or Elastic stroke
  • • Create account on CODEPEN and follow different people who creates different things using HTML, CSS and JavaScript.
